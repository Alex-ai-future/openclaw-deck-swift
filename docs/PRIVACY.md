# 隐私政策

**最后更新：** 2026 年 3 月 3 日

OpenClaw Deck Swift（以下简称"本应用"）是一个开源的 AI 聊天客户端。本隐私政策说明我们如何处理数据。

## 1. 数据收集

**本应用不收集、不存储、不传输任何用户数据。**

- 所有聊天记录、Session 信息、配置数据均存储在**用户设备本地**
- 本应用没有后端服务器，不收集任何分析数据或使用追踪器

## 2. Cloudflare 同步功能

本应用提供可选的 Cloudflare 同步功能，允许用户在多设备间同步配置和聊天记录。

**重要说明：**
- Cloudflare 账户由**用户自行提供和配置**
- 数据存储在**用户自己的 Cloudflare R2/KV 存储空间**中
- 开发者**无法访问、控制或查看**任何存储在 Cloudflare 的数据
- 用户可以随时在应用设置中禁用同步功能

## 3. Gateway 连接

本应用需要连接 OpenClaw Gateway 才能工作：
- Gateway 由用户自行部署和配置
- 所有与 Gateway 的通信均在用户控制的设备间进行
- 开发者不运营、不控制任何 Gateway 服务器

## 4. 第三方服务

本应用**不使用**任何第三方分析服务、广告 SDK 或数据追踪器。

## 5. 数据来源

本应用是**开源软件**，源代码完全公开：
- GitHub: https://github.com/Alex-ai-future/openclaw-deck-swift
- 任何人都可以审查代码，验证数据处理方式

## 6. 儿童隐私

本应用不面向 13 岁以下儿童，也不故意收集儿童数据。

## 7. 隐私政策变更

如有重大变更，我们会通过 GitHub 仓库公告。继续使用本应用视为接受变更。

## 8. 联系方式

如有隐私相关问题，请通过以下方式联系：
- GitHub Issues: https://github.com/Alex-ai-future/openclaw-deck-swift/issues

---

**核心原则：** 你的数据，你做主。我们只是提供了一个工具，但完全不接触你的数据。
